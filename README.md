# scr2spr
screen to sprite converter fo ZX-Spectrum

coordinates and sizes sets in char

usage:
'-x', '--x', default=0, type=int)
'-y', '--y', default=0, type=int)
'-len', '--length', type=int)
'-hei', '--height', type=int)
'-c', '--count', default=1, type=int)
'-i', '--input', type=str)
'-o', '--output', default='sprite.bin', type=str)
